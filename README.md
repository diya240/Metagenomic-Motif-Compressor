# Metagenomic-Motif-Compressor
Metagenomic Motif Compressor is a dictionary-based lossless genomic data compression system that identifies recurring DNA motifs and replaces them with compact reference identifiers, reducing storage requirements while preserving the original sequence for accurate reconstruction and analysis.

## Author
Diya Chakraborty

## Project Overview
Metagenomic Motif Compressor is a dictionary-based lossless genomic data compression system that identifies recurring DNA motifs and replaces them with compact reference identifiers.

## Objectives

- Detect repeated DNA motifs
- Create a motif dictionary
- Compress DNA sequences
- Ensure lossless reconstruction
- Evaluate compression performance
